vim 打开 可以tab补全

i键 文本里进行编写模式

ESC 退出编写

shift+：wq 保存并退出 w 保存 q退出

mkdir 创建文件夹

touch 创建文件

rm 删除文本

rm -r 删除文件夹

mv 改名或者移动

cp 复制

cat 将文件内容显示到终端

pwd 显示目前所在位置目录

cd 切换路径

ls 列出所在文件夹的文件

ls -l 显示出用户的所有内容

cd .. 返回上一级文件

cd ../.. 返回上面两层

wget 下载文件

ip addr 查看ip地址

clear 清屏命令

df -h 查看分区对应的目录

命令提示符，显示了

  1. 当前登录的用户名
  2. 用户当前所在的路径

文件系统结构：

最大目录/ 依次往下，为了防止系统写满开不了机，/boot与/home单独写,也都是/的子目录，都有自己独立的空间。

总结：

  • 分区：把大硬盘分为小的逻辑分区
  • 格式化：写入文件系统
  • 分区设备文件名：给每个分区定义设备文件名
  • 挂载：给每个分区分配挂载点，挂载点必须是目录，而且这个目录必须是空目录才可以挂载点分配
